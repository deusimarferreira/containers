## rkt
Como instalar o rkt?

```sh
# Clone código fonte a partir do repositório GitHub
$ git clone https://github.com/coreos/rkt

$ cd rkt

# Execute o script para realizar a instalação
$ ./scripts/install-rkt.sh
  ```
